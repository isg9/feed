# advogato plugin for wordpress 1.5 — wingolog

## [advogato plugin for wordpress 1.5](/archives/2005/04/10/advogato-plugin-for-wordpress-15)

11 April 2005 0:36 AM

- [hacks](/tags/hacks)

I updated my [advogato plugin for wordpress](//wingolog.org/pub/advogato.php.txt) to work with wordpress 1.5.

Besides that, I added a piece of meta-data on the wordpress posts so that when you edit a post in wordpress, it will edit the proper post on advogato. Also, there are fun [elanthis](http://advogato.org/person/elanthis/)-style permalinks at the bottom ;)

See also the [advogato importer for wordpress](//wingolog.org/archives/2004/12/30/advogato-importer-for-wordpress).
