Friends, [I'm speaking at JSConf.us this year](http://2012.jsconf.us/)! Yee haw!

I would have mentioned this later, but [events push me to say something now](http://twitter.com/#!/jsconf/status/171992937742344194).

You see, I wrote the [web server that runs this thing](http://www.gnu.org/software/guile/manual/html_node/Web.html#Web), together with the [blog software](//wingolog.org/software/tekuti). I've been hacking on it recently, too; more on that soon. But it seems that this is the first time I've noticed a link from a site that starts with a number. The URI parsers for the referer link were bombing out, because I [left off a trailing `$` on a regular expression](http://git.savannah.gnu.org/cgit/guile.git/commit/?h=stable-2.0&id=1868309a9e34a04a5b3020e147d0ce029038b290).

So, for love and `$`, JSConf ho! We ride at dawn!
