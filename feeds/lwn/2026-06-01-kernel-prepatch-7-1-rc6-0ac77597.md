# Kernel prepatch 7.1-rc6

The [7.1-rc6](https://lwn.net/Articles/1075574/) kernel prepatch is out for testing. Linus said: "Well, I wouldn't call this 'small', but it is certainly smaller than rc5 was. And I don't think there's anything particularly scary here, so maybe we're still on track for a normal release cycle. Let's see."
