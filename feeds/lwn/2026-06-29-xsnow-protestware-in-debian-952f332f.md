# [$] Xsnow "protestware" in Debian

The [xsnow](https://www.ratrabbit.nl/ratrabbit/xsnow/index.html) application, which generates an animated snowfall effect (and other pleasant diversions) for X11 desktops, does not seem like an obvious channel for political statements. Nevertheless, xsnow's maintainer seems to have included a political protest in the program: an Easter egg that is triggered when the program's language is set to Russia ("ru"). One user has complained that this functionality should be removed from the Debian [xsnow
package](https://packages.debian.org/STABLE/games/xsnow), but Debian does not seem to have any rules that forbid such a feature outright.
