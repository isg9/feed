# Git 2.55.0 released

Git maintainer Junio Hamano has [announced](https://lwn.net/ml/all/xmqqv7b1w9vr.fsf%40gitster.g/) Git 2.55.0, which has non-merge commits from 100 people; 33 of those are first-time contributors to the project. LWN [recently covered](https://lwn.net/Articles/1079596/) some of the noteworthy changes in 2.55, including new features for the experimental " `git history`" command, addition of the Git [fsmonitor](https://www.man7.org/linux/man-pages/man1/git-fsmonitor--daemon.1.html) daemon for Linux systems, and more.
