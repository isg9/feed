# News

To make FogBugz work on Unix as well as Windows, we needed a PHP version. Rather than do a one-time port, we built a compiler that automatically generates a PHP version from the ASP source code. Read all about it in today’s [part III of The Road to FogBugz 4.0](https://www.joelonsoftware.com/articles/FogBugzIII.html).
