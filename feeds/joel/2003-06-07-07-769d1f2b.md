# 07

![](https://i0.wp.com/www.joelonsoftware.com/wp-content/uploads/2003/06/07Dweep.png?resize=35%2C34&ssl=1)Steve Pavlina has written up some [great business tips](http://www.dexterity.com/articles/) for shareware and games developers.

[Steven Den Beste](http://denbeste.nu/cd_log_entries/2003/06/Aheadofthewave.shtml): “Citydesk cannot be beaten.” \[Well, not without knowing the cheat codes.\]

Ellen Ullman’s new book [The Bug](http://www.amazon.com/exec/obidos/tg/detail/-/0385508603/joelonsoftware) is a terrific novel about a programmer coming unravelled because of a killer bug nicknamed *The Jester* which seems to only appear at the worst possible times. Highly recommended. (There’s a bug in the book, too, probably left in as a reward for real programmers. I don’t want to spoil it for you so I’ll describe it in an HTML comment right here.)
