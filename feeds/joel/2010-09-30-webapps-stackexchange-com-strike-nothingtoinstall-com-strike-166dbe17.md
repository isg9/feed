# WebApps.stackexchange.com <strike>nothingtoinstall.com</strike> now live!

The first Stack Exchange site to make it all the way through the community creation process is now live and out of beta!

[![Nothing to Install screenshot](https://i0.wp.com/www.joelonsoftware.com/wp-content/uploads/2010/09/30NothingToInstall.png?w=730&ssl=1)](http://nothingtoinstall.com/)

[webapps.stackexchange.com](http://webapps.stackexchange.com/) is a place to get help with web applications. Want to know [how to email huge files](http://nothingtoinstall.com/questions/1062/how-to-send-big-files-via-email)? How to [delete your Facebook account](http://nothingtoinstall.com/questions/1/how-do-i-delete-my-facebook-account)? How to [secretly follow someone on Twitter](http://nothingtoinstall.com/questions/1178/how-can-i-follow-someone-on-twitter-without-them-knowing)? [How to backup WordPress blogs](http://nothingtoinstall.com/questions/411/how-can-i-backup-my-wordpress-com-hosted-blog), ahem, Jeff? This is the site.

( **Updated:** Only days after launching, we changed the name from “nothingtoinstall.com” to “webapps.stackexchange.com” because nobody really liked that name!)
