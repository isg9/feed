# Stack Overflow Podcast #27

[![](https://i0.wp.com/www.joelonsoftware.com/wp-content/uploads/2008/08/StackOverflow.png?w=730&ssl=1)](http://blog.stackoverflow.com/2008/10/podcast-27/) Our guests on this week’s [Stack Overflow Podcast](http://blog.stackoverflow.com/2008/10/podcast-27/) are the founders of Reddit, Steve Huffman and Alexis Ohanian.

By the way, Jeff recently upgraded the database server from Microsoft SQL Server 2005 to 2008, and found pretty conclusively that 2008 has a new architecture for full text search which is [significantly slower](http://blog.stackoverflow.com/2008/11/sql-2008-full-text-search-problems/) than it was in 2005. Something to be careful about if you’re thinking of upgrading to 2008.
