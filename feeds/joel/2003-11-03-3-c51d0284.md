# 3

[![New Fog Creek Home Page](https://i0.wp.com/www.joelonsoftware.com/wp-content/uploads/2003/11/03fc.jpg?resize=200%2C219&ssl=1)](http://www.fogcreek.com) Fog Creek’s website has been [redone](http://www.fogcreek.com) to use our new Sam Sherwood-designed logo. The cutting edge page design is thanks to superstar web designer [Dave Shea](http://www.mezzoblue.com/), famous for the [CSS garden](http://www.csszengarden.com/) and the eye-popping [new Mozilla home page](http://website-beta.mozilla.org/), with additional programming and graphics by Fog Creek’s own Dmitri Kalmar. It’s about 99% standards-compliant (with the exception of a couple of stray FONT tags left over from old content that hasn’t been updated… oh the horror!).
