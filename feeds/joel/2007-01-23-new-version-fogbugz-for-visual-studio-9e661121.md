# New version: FogBugz for Visual Studio

I fixed a bunch of little bugs in [FogBugz Add-in for Visual Studio](http://www.fogcreek.com/FogBugz/KB/setup/InstallFogBugzforVisualSt.html). The new version, 2.1, is now available.

![](https://i0.wp.com/www.joelonsoftware.com/wp-content/uploads/2006/12/17FBforVS.png?w=730&ssl=1)

It now remembers its position when you exit Visual Studio, and works better with SSL and authenticated FogBugz installations. [A complete list of bug fixes is here](http://www.fogcreek.com/FogBugz/KB/setup/VersionHistoryofFogBugzfo.html).

Thanks to Olivier, James, Matt, Jason, Alan, and Miguel for detailed bug reports which helped me track down and fix all these problems.
