# Regular Expression Article \#4 Russ Cox January 19, 2012 *research.swtch.com/regexp4* Posted on Thursday, January 19, 2012.

In January 2007 I posted an article on my web site titled “ [Regular Expression Matching Can Be Simple And Fast.](http://swtch.com/~rsc/regexp/regexp1.html)” I intended this to be the first of three; the second would explain how to do submatching using automata, and the third would explain how to make a really fast DFA. These were inspired by my work on Google Code Search.

Today, the [fourth article](http://swtch.com/~rsc/regexp/regexp4.html) in my three-part series is available, accompanied by source code (as usual). This one describes how Code Search worked.
