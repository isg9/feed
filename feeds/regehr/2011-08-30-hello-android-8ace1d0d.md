# Hello Android

Some semesters I teach courses that just need to be taught. On the other hand, this Fall I get to teach a class that should be purely fun — an Android projects course for upper-level undergrads. I already promised an “A” to anyone who (legitimately) makes at least $100 using an application developed for the class. I plan to nudge students away from web applications and single-node games, which seem boring. Rather, I hope they’ll interface with the awesome peripherals that make phones and tablets special. Today in class we looked through the source code for a [pedometer app](http://code.google.com/p/pedometer/) — the most interesting part being the step detection logic. It turned out to be pretty hacky, but it more or less works.

![](https://blog.regehr.org/wp-content/uploads/2011/08/2011-08-30-13.33.03.jpg)

Here’s a gratuitous hardware shot — my pile of Galaxy 10.1 tabs, which the students got to take home today. Half of the tabs were gifts, half were purchased using university funds acquired by [Thomas Schmid](http://www.ece.utah.edu/~schmid/) who was awesome enough to loan them to me (or rather, to my students) for the Fall. The most interesting part of the class will be when I get some Tegra 3 development boards — these should be seriously awesome.
