# Some thoughts on Quora

With the announcement this week that Quora had taken $11 million in VC at an $86 million valuation, there’s been an awful lot of attention on Quora. I’ve had an account there and wanted to write up some of my initial thoughts. If you haven’t heard about Quora, it’s yet another question/answer site on the web. People pose questions, and you can view questions and answer them. I’ve heard it described as “StackOverflow, but for anything”, which is roughly true, even if I think they want to be more.
